## Devlopment

### Prerequisites

- Rust - https://rustup.rs/
- Trunk - https://trunkrs.dev/ (only needed for web)

### Running

Native hot reload:
```
cargo watch -x run --features bevy/dynamic
```

Web hot reload:
```
trunk serve
```
